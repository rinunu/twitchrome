ツイッタークライアント 開発 ver.

本アプリは Chrome ブラウザ上で動く Twitter クライアントです。 「使っていて楽しい」をコンセプトに開発中です。

  • 連絡先: twitter
  • ソース: github
ログインページ

対応ブラウザ

  • Chrome7(対応済み)
  • Safari4(対応済み)
  • Firefox4(対応予定)
  • IE9(対応予定)

主な特徴

  • よく話すユーザへのショートカットを表示します
  • 「戻る/進む」にて直前/直後のタイムラインへ移動できます
  • リプライチェーンを表示します
  • 返信の付いたツイートをアイコンで識別できます
  • 画像をインラインで表示します(現在 Twitpic のみ対応)
  • 他のユーザのタイムラインを表示すると、そのユーザの背景画像を表示します
